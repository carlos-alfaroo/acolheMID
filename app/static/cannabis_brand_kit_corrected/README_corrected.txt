Qué ajusté para que sea más fiel:
- Mantengo la forma de cuadro redondeado con borde interior.
- Hoja centrada con tres foliolos estilizados (central + laterales rotadas).
- La cruz médica integrada en la "cola" inferior del marco, con conector rectangular similar a un "plug" para transmitir cuidado/servicio médico.
- Proporciones optimizadas para que el icono pueda usarse solo o con wordmark horizontal/vertical.

Cómo editar el texto 'TuMarca':
- Edita el elemento <text> dentro del SVG con Figma, Illustrator o un editor de texto.
- Para web, carga Poppins/Inter desde Google Fonts para preservar el kerning.

Si quieres, reemplazo 'TuMarca' por el nombre real y te genero versiones en PNG (varios tamaños) para redes y favicon HTTP listos. También puedo ajustar la paleta (matices de verde), el grosor de trazo o la integración del símbolo médico para que sea más sutil o más explícito — lo que prefieras.
