
# Mini Brand Kit — TuMarca (Cannabis medicinal)

Palette (Tailwind):
- green-900: #14532d (primary)
- green-200: #bbf7d0 (tint for dark bg)
- white: #ffffff (light bg)

Typography:
- Headlines/Wordmark: Poppins 700
- UI/Text: Inter 400–600
Both available on Google Fonts.

Files:
- verdacare-logo-icon-dark.svg — icon for dark UIs or green-900 backgrounds
- verdacare-logo-icon-light.svg — icon for light backgrounds
- verdacare-logo-wordmark-horizontal-dark.svg — icon + wordmark horizontal
- verdacare-logo-wordmark-stacked-light.svg — stacked layout for posters/social
- verdacare-favicon.svg — simplified icon

Guidelines:
- Minimum clear space: height of the medical cross arm around the icon.
- Minimum icon size for web: 24px (use favicon); preferred 32–48px+.
- Don't stretch the logo; keep aspect ratios. Only use approved colors above.
- On photographs, place the dark icon over a semi-transparent light overlay.

How to replace 'TuMarca':
- Open the SVG in a vector editor (Figma/Illustrator) and edit the <text> nodes.
- Fonts will fall back to system sans; to preserve look, load Poppins/Inter on web.
